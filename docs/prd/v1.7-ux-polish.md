# v1.7 UX Polish — Autocomplete, Context Menu, Reactions, Tool Display

## Status
Approved

## Features

### F1 — Autocomplete for /model and /agent use
Discord autocomplete shows suggestions as user types.
- `/model` → suggest: sonnet, opus, haiku
- `/agent use` → suggest from agent_manager.list_all()
- Use `@app_commands.autocomplete`

### F2 — Context Menu: "Send to Claude"
Right-click a message → "Send to Claude" → creates thread with that message as prompt.
- `@app_commands.context_menu(name="Send to Claude")`
- Works on any message in a bound channel

### F3 — Message Reactions Status
- User sends message → bot adds ⏳ reaction
- Claude starts responding → change to 🔄
- Claude finishes → change to ✅
- Error → ❌

### F4 — WebSearch/WebFetch/Glob/Grep Display
Specialized embeds for these tools:
- WebSearch: 🔍 embed with query + result count
- WebFetch: 🌐 embed with URL
- Glob: 📂 embed with pattern + matched files
- Grep: 🔎 embed with pattern + matches

### F5 — Session Auto-Expiry
- Sessions inactive for >1 hour auto-disconnect
- Saves session_id for resume
- Background task checks every 5 minutes
- Configurable via CLAUDED_SESSION_TIMEOUT env var

### F6 — Modal for System Prompt
- `/project system-prompt` opens a Modal with a paragraph text input
- Better UX for long prompts than command parameter

### F7 — PostToolUse + Stop Hooks
- PostToolUse: update tool embed with execution time
- Stop: send session summary embed (total cost, turns, duration)

### F8 — Environment Variables
- `/project env set KEY VALUE` — set env var for project sessions
- `/project env list` — list vars (values masked)
- `/project env remove KEY`
- Passed via ClaudeCodeOptions(env=...)

### F9 — Export Session
- `/session export` → generates markdown file of conversation history
- Uploads as .md file attachment

### F10 — Worktree/Cron Display
- EnterWorktree/ExitWorktree → embed showing worktree name
- CronCreate/Delete/List → embed showing scheduled tasks
